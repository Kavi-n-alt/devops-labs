# Task 2 – CI/CD Pipeline Using GitHub Actions

This task implements a simple CI pipeline that runs automatically when code is pushed to the main branch.

## Workflow steps:
- Checkout repository
- Set up Python
- Install dependencies
- Syntax check of Flask app
- Docker image build

## How to view pipeline:
1. Go to GitHub repo → Actions tab
2. Click "CI Pipeline"
3. View logs and take screenshots
